# rootnode-website

Source for [rootnode.design](https://rootnode.design) — the marketing surface for [root.node](https://github.com/drayline/rootnode-skills).

Static single-page HTML deployment via Cloudflare Pages. No build pipeline.

## Stack
- HTML, inline CSS, inline JS
- Google Fonts (Inter, JetBrains Mono)
- Cloudflare Pages (auto-deploy on push to `main`)
